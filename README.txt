
LEEME
Bienvenido a Nuestro Encriptador de Palabras
Gracias por visitar nuestro sitio web. Este proyecto ha sido desarrollado con JavaScript y tiene como objetivo proporcionar una herramienta sencilla y efectiva para encriptar y desencriptar palabras.

Descripción del Proyecto
Nuestra aplicación permite a los usuarios encriptar y desencriptar palabras de manera segura. Utilizamos algoritmos de encriptación para transformar texto legible en un formato cifrado, asegurando que la información sensible se mantenga protegida. De igual manera, ofrecemos la capacidad de revertir este proceso, permitiendo que las palabras encriptadas vuelvan a su forma original.

Características
Encriptación de Palabras: Introduce cualquier palabra y nuestro sistema la convertirá en un formato encriptado, protegiendo así su contenido.
Desencriptación de Palabras: Ingresa una palabra encriptada previamente utilizando nuestra herramienta y obtén la versión original.
Instrucciones de Uso
Para Encriptar:

Escribe la palabra que deseas encriptar en el campo correspondiente.
Haz clic en el botón "Encriptar".
La palabra encriptada se mostrará en pantalla, lista para copiar y usar.
Para Desencriptar:

Escribe la palabra encriptada en el campo designado.
Haz clic en el botón "Desencriptar".
La palabra original aparecerá en pantalla.
Tecnologías Utilizadas
Este encriptador de palabras está construido con JavaScript, aprovechando su capacidad para manejar operaciones en tiempo real y su flexibilidad para integrarse en diversos entornos web.

Objetivo del Proyecto
El objetivo de esta herramienta es facilitar la protección de información textual, permitiendo a los usuarios encriptar datos sensibles y desencriptarlos cuando sea necesario, todo a través de una interfaz sencilla e intuitiva.
